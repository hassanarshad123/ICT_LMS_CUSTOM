# Scripts

Database seeding and utility scripts. Run from backend/ with venv activated.

- `seed_super_admin.py` — Create super_admin user (uses env vars)
- `seed_data.py` — Full sample data for development
- `seed_a1_data.py` — A1 institute sample data
- `seed_ict_data.py` — ICT institute sample data
- `generate_secret.py` — Generate JWT secret key
