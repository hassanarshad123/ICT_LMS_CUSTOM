# Components

UI components organized by type and user role.

- `layout/` — Sidebar, header, dashboard wrapper (role-based navigation)
- `dashboards/` — Role-specific dashboard views (admin, CC, teacher, student)
- `pages/admin/` — Admin page components (monitoring, branding, users, etc.)
- `pages/cc/` — Course creator components (upload, quiz builder, batch management)
- `pages/student/` — Student components (course detail, quiz taking)
- `pages/teacher/` — Teacher components (course detail, classes)
- `shared/` — Reusable components (video player, error boundary, notifications, CSV import)
- `ui/` — Shadcn/Radix primitives (button, dialog, input, select, etc.)
