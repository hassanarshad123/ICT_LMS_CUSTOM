# Hooks

Custom React hooks for data fetching and real-time features.

- `use-api.ts` — useApi (GET) and useMutation (POST/PATCH/DELETE) with TanStack Query.
  Supports keepPreviousData and targeted cache invalidation.
- `use-paginated-api.ts` — Paginated queries with page state management
- `use-websocket.ts` — WebSocket hooks for class status, announcements, notifications
- `use-base-path.ts` — Get current user's base URL path
- `use-toast.ts` — Sonner toast wrapper
