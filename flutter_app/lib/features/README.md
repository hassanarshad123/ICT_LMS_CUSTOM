# Features

Feature modules organized by user function. Each has screens/, widgets/, and optionally providers/.

- `auth/` — Login, forgot password
- `onboarding/` — Institute slug entry
- `home/` — Student dashboard with stats, batches, upcoming classes
- `courses/` — Course list, detail, lecture player
- `classes/` — Live classes list, recordings
- `quizzes/` — Quiz start, taking, results (swipe card UI)
- `notifications/` — Notification list
- `profile/` — Profile, settings, certificates, jobs, announcements
- `search/` — Global search
